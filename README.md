# My Online Portfolio & CV
This project is consisted of two parts: **an online portfolio generator** and **a CV generator**.

1. The online portfolio is based on [Jinja2](https://jinja.palletsprojects.com) and [TOML](https://toml.io). This part is heavily influenced by  [Corey Schafer's static website tutorial](https://youtu.be/AD-3nVI3-_U). Instead of using json files, in this project TOML files are been adopted.

2. The CV is written by [Typst](https://typst.app). This part is inspired by [Rendercv](https://rendercv.com).
